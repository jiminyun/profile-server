# My-Profile

Resume + Portfolio management with MERN stack

## Built With

- Frontend
  - JavaScript(ES6+)
  - React(JSX, Redux, Ducks, Redux-thunk)
  - Sass
  - Netlify
- Backend
  - Mongo DB
  - Node.js/Express
  - EC2, S3

## Getting Started

```
git clone https://github.com/jiminyun/profile-serever

npm install
yarn dev:server
```
